# Redis

## 介绍
   
我的Redis学习主要是参考《Redis设计与实现》、《Redis深度历险》、《Redis开发与运维》这三本书籍，Redis设计与实现讲解内容非常详细，而且文中有大量的代码案例与流程图示，对Redis初级体系架构的养成是非常不错的一本书籍。深度历险书中介绍了非常深刻的知识点，可作为知识的补充。而开发与运维广度非常全面，可以结合设计与实现来看。
